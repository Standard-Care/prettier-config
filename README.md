# `prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm install https://github.com/Standard-Care/prettier-config --save-dev
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "https://github.com/Standard-Care/prettier-config"
}
```
