# `prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm install --include=dev https://github.com/Standard-Care/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "https://github.com/Standard-Care/prettier-config"
}
```
